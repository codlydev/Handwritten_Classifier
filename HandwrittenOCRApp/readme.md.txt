# Handwritten OCR App

A simple web-based application that performs Optical Character Recognition (OCR) on uploaded handwritten images. Built using Python and Flask, the app uses Tesseract OCR to extract text from images and display the results in a user-friendly web interface.

---

## 🧠 Features

- Upload handwritten image files via a web browser
- Automatically extract and display the recognized text
- Clean and responsive frontend UI
- Real-time logging of OCR operations
- Easy to run locally
