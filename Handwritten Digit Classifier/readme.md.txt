# 🧠 Handwritten Digit Classifier (MNIST)

A simple neural network built with TensorFlow and Keras to recognize handwritten digits from the MNIST dataset. This project includes data visualization, model training, evaluation, prediction, and saving the trained model.

---

## 📂 Project Structure




---

## 🚀 Features

- Loads and preprocesses MNIST digit data
- Visualizes sample digits
- Builds and trains a simple neural network
- Evaluates model performance
- Predicts digits from test data
- Saves the trained model as `.h5`
